The capstone project folder contains the following files.
1) Capstone proposal
2) Capstone project report describing the datasets, solution, evaluation. 
3) Capstone project code in python
4) Readme file

Installation steps:

Install pandas, quandl, sklearn, matplotlib using the following commands in anaconda prompt.

"conda install -c anaconda quandl" or "pip install quandl"

"python -m pip install pandas"

"pip install -U scikit-learn"

"python -m pip install matplotlib"



Execute the code in anaconda by:
 
python capstone "quandl code"

give quandl code in double quotes.
